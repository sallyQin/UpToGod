# UpToGod
《听天由命》是一款帮助“选择障碍者“快速地做出选择的小应用。
## * 实现3个主要功能：
####  1. 可通过设置旋转的方向、转速以及摩擦力来控制转盘旋转。
####  2. 实现自定义背景、圆框、指针的设置。
####  3. 通过对问题和选择进行数据的增、删、改操作。
